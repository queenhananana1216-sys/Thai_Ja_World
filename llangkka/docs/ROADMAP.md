# llangkka 확장 로드맵

## 1단계: 터미널 최적화(완료 목표)
- PowerShell 7 : PSReadLine 히스토리 검색/예측 + starship 프롬프트
- WSL Ubuntu : bash 히스토리 + starship 프롬프트
- Windows Terminal : `llangkka` 프로필 추가 템플릿 제공

## 2단계: 오프라인 번역
- 지원 언어 1순위 확정
- 데이터 다운로드(SSD 고정) + 캐시 우선 사용

## 3단계: 오프라인 GPT
- 로컬 LLM 실행 방식(예: Ollama vs llama.cpp) 확정
- 모델 다운로드(SSD 고정) + 오프라인 실행
