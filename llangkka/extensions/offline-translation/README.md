# 오프라인 번역 확장(구조만 제공)

요구사항: SSD에 특정 나라 글자/언어 데이터(사전/모델/폰트)를 받아서 네트워크 없이도 번역이 가능하도록 준비.

## 폴더 구조(의도)

- `data/` : 번역에 필요한 언어별 리소스(사전/모델/캐시 등)
- `fonts/` : 해당 언어가 깨지지 않도록 폰트 캐시(필요 시 설치/캐시)

## 다음에 정해야 할 것(질문)

1. 어떤 “특정 나라/언어”를 우선 지원할까요? (예: 태국어, 라오스어, 베트남어 등)
2. 번역은 어떤 방식이 목표일까요?
   - 오프라인 모델 기반(로컬 LLM/번역 모델)
   - 사전/규칙 기반(오프라인 사전)

답 주시면, 그 방식에 맞춰 다운로드/설치 스크립트를 이 폴더에 추가하겠습니다.
