SSD(E:) 기준 — 프로젝트·에셋 정리 템플릿 (태자 월드 포함 공통)
================================================================================
Cursor에서 폴더 열기: File → Open Folder → 작업 중인 루트 선택

권장 루트 예시:

  E:\Dev\
    thai-ja-world\          ← Next 앱 (현재: 이 폴더 02_Workspace 를 나중에 여기로 통일 가능)
    <다른프로젝트>\

  E:\Assets\
    thai-ja-world\
      figma\
      brand\

  E:\Archive\
    구버전 백업 (DB 덤프는 접근 통제)

운영 규칙:
  - 코드는 Git 한 루트. .env.local 은 커밋 금지.
  - Supabase SQL 정본: 레포 supabase\ 와 대시보드 중 하나로 통일.
  - thaijaworld.com 은 호스팅 환경변수·Redirect URL 을 표로 관리.
